# studio-adventures
Studio Adventures — printable solo RPGs and micro story artifacts. This repository hosts free downloadable PDF adventures designed to be printed, folded, and played offline.


**Studio Adventures**

A growing collection of printable solo RPG artifacts.

These adventures are designed to be simple, physical, and playable offline.
Print on A4. Fold. Grab a pencil. Begin.

Current Releases
Studio Adventure 03 — Garden Escape

A solo micro-adventure about survival in a hostile garden.
Play time: ~15–20 minutes.

Studio Rules — Core System

Lightweight rules used across Studio Adventures.
Compatible with future releases.

How to Play

Download the PDF.

Print on A4 (standard settings are fine).

Fold as indicated.

Play solo.

No apps. No accounts. No tracking.
Just paper and imagination.
